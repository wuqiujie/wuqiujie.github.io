PROJECT / VIRTUAL FILM SET

Virtual Film Set

An adaptive assisting interface that helps learners practice virtual cinematography in VR.

Programmer · Designer HTC Vive · VR 2019–2020 Research · WICED / Eurographics
Original Wix Page ↗ Paper ↗ Conference Talk ↗

Filmmaking practice inside a virtual film set with adaptive guidance.

Overview

Virtual Film Set is an experimental learning environment that teaches filmmaking skills inside VR. Learners operate a virtual film set with lights, cameras and actors, while an assisting interface provides contextual hints through visuals and audio. The system adapts to each learner’s performance and experience level, with the goal of supporting practice without taking away agency or creativity.

Role & Responsibilities

  • Designed the user model describing how filmmakers progress from beginner to mastery.
  • Implemented the assisting layer, including hint logic, visual overlays and sound prompts.
  • Integrated the interface into an existing virtual film set pipeline.
  • Designed and ran a controlled user study with 24 participants, and analysed both quantitative results and open-ended feedback.

Scaffolding & Adaptive Design

The design combines VR cinematography with scaffolding theory to provide step-by-step support that gradually fades as learners become more confident.

The assisting interface is inspired by educational scaffolding: learners receive more structured guidance at the beginning, and the level of support decreases as they demonstrate higher mastery. Hints can highlight areas of interest in the set, suggest camera positions, or remind the learner about shot composition and continuity.

System Architecture & Hint Types

The system tracks learner actions through a user model spanning four levels: beginner, intermediate, advanced and mastery. Based on this model it chooses different hint types, ranging from explicit arrows and overlays to subtler cues that preserve creative freedom.

Adaptive Mechanism & Study

The adaptive mechanism combines automatic monitoring with explicit feedback from the learner. Participants can ask for more or fewer hints, and the system updates their proficiency level accordingly. In the study, participants using the adaptive interface reported smoother learning experiences and showed stronger gains on follow-up knowledge questions compared to those using the film set without assistance.

User Feedback

Participants described the interface as supportive and reassuring, especially when they were unsure about how to set up lights or framing. Many felt the guidance struck a good balance between teaching and letting them experiment on their own.
